# 📖 Alpha Blog <a name="about-project"></a>

**Alpha Blog** is a ruby on rails web application integrated with Bootstrap for the front end styling. It has all the functionalities such as making a post, editing and deleting a post.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <ul>
    <li><a href="https://rubyonrails.org/">Ruby on Rails</a></li>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
    <li><a href="https://getbootstrap.com/">Bootstrap 5</a></li>
  </ul>

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Create a post**
- **Edit a post**
- **Delete a post**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

    - ruby
    - ruby on rails
    - postgreSQL

### Setup

Clone this repository to your desired folder:

commands:

```sh
  cd my-folder
  git clone https://github.com/Log-benjamin/Alpha-Blog
```

### Update your database username and passwrord
```sh 
   cd config/database.yml
    username: your local machine username
    password: your local machine password
```
### Install

Install this project with:
command:

```sh
  cd my-project
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```
